# 2.5D_ERT_Cylinder
This program employs the Modified Optimization Wavenumbers for 2.5D electrical resistivity forward modeling of cylindrical objects, along with experimental data sets and relevant error comparison calculation programs.
The help.txt file in the example_forwardmodeling folder provides a running example of the forward modeling program, while the help.txt file in the inversion folder gives a running example of the inversion program.
